# Build

## Prerequisites

 - [io.js](https://iojs.org/en/index.html) *or Node.js with support for template strings (not currently avaialable)*

 ## Instructions:

When developing run the following to build:

`npm run build-dev`

To bundle for production, run:

`npm run build`
